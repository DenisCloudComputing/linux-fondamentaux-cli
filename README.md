 Linux — Fondamentaux CLI

> Mes travaux pratiques sur les commandes fondamentales de Linux,
> réalisés dans le cadre de ma formation en Cloud Computing.

---

 Description

Ce repository regroupe mes labs d'apprentissage des commandes
de base Linux en ligne de commande (CLI). Chaque lab est documenté
avec les commandes pratiquées, les explications et des captures d'écran.

---

 Objectifs d'apprentissage

- Maîtriser la navigation dans le système de fichiers Linux
- Manipuler les fichiers et répertoires en ligne de commande
- Comprendre la structure de l'arborescence Linux
- Utiliser les commandes essentielles au quotidien

---

 Technologies utilisées

| Technologie | Usage |

| Bash | Shell / Interpréteur de commandes |
