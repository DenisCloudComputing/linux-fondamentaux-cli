 Lab 01 — Navigation dans le système de fichiers

 Objectif
Maîtriser les commandes de navigation de base dans Linux.

 Commandes pratiquées

| Commande | Description | Exemple |
|----------|-------------|---------|
| `pwd` | Afficher le répertoire courant | `pwd` → `/home/denis` |
| `ls` | Lister le contenu d'un dossier | `ls -la` |
| `cd` | Changer de répertoire | `cd /var/log` |
| `tree` | Afficher l'arborescence | `tree -L 2` |
| `clear` | Effacer le terminal | `clear` |

 Notes importantes
- `cd ..` → remonter d'un niveau
- `cd ~` → retourner au répertoire personnel
- `cd -` → retourner au répertoire précédent
- `ls -la` → afficher TOUT, y compris les fichiers cachés
