 Aide-Mémoire — Commandes Linux CLI

 Navigation
| Commande | Action |
|----------|--------|
| `pwd` | Où suis-je ? |
| `ls` | Qu'est-ce qu'il y a ici ? |
| `ls -la` | Tout montrer (même les cachés) |
| `cd chemin` | Aller quelque part |
| `cd ..` | Remonter d'un niveau |
| `cd ~` | Retourner à la maison |
| `cd -` | Retourner d'où je viens |
| `tree` | Voir l'arborescence |

 Manipulation de fichiers
| Commande | Action |
|----------|--------|
| `touch fichier` | Créer un fichier vide |
| `mkdir dossier` | Créer un dossier |
| `mkdir -p a/b/c` | Créer des dossiers imbriqués |
| `cp source dest` | Copier |
| `cp -r source dest` | Copier un dossier |
| `mv source dest` | Déplacer ou renommer |
| `rm fichier` | Supprimer un fichier |
| `rm -r dossier` | Supprimer un dossier |

 Lecture de fichiers
| Commande | Action |
|----------|--------|
| `cat fichier` | Tout afficher |
| `head fichier` | Les 10 premières lignes |
| `tail fichier` | Les 10 dernières lignes |
| `less fichier` | Naviguer dans le fichier |
| `wc -l fichier` | Compter les lignes |

 Recherche
| Commande | Action |
|----------|--------|
| `find / -name "fichier"` | Chercher par nom |
| `grep "mot" fichier` | Chercher du texte |
| `which commande` | Où est installée la commande |
| `locate fichier` | Recherche rapide (base indexée) |
